Program pentru afișarea tuturor numerelor impare dintr-un interval

3 variante: Python (începători), Python (funcții), C++ (începători).

Python Începători

Codul este păstrat identic cu exemplul oferit.

Cod complet Exact ca în exemplu 
lower=int(input("Enter the lower limit for the range:"))
upper=int(input("Enter the upper limit for the range:"))
for i in range(lower,upper+1):
    if(i%2!=0):
        print(i)

Program Explanation (tradus)

1. Utilizatorul trebuie să introducă limita superioară și limita inferioară ale intervalului.
2. Bucla for parcurge valorile de la limita inferioară până la limita superioară.
3. Expresia din instrucțiunea if verifică dacă restul obținut la împărțirea numărului la 2 este egal cu 1 sau nu (folosind operatorul %).
4. Dacă restul nu este egal cu 0, numărul este impar și, prin urmare, este afișat.

Observație (programare corectă): în practică se verifică dacă lower ≤ upper. Dacă utilizatorul introduce limitele invers (lower > upper), se poate fie interschimba valorile, fie afișa un mesaj de eroare. (Codul de mai sus rămâne neschimbat.)

Python Funcții

Variantă organizată cu funcție.

Cod complet Funcție + apel 
def print_odd_numbers_in_range(lower, upper):
    for i in range(lower, upper + 1):
        if i % 2 != 0:
            print(i)

lower = int(input("Enter the lower limit for the range:"))
upper = int(input("Enter the upper limit for the range:"))
for i in range(lower,upper+1):
    if(i%2!=0):
        print(i)

Explicația programului

1. Utilizatorul trebuie să introducă limita superioară și limita inferioară ale intervalului.
2. Bucla for parcurge valorile de la limita inferioară până la limita superioară.
3. Expresia din instrucțiunea if verifică dacă restul obținut la împărțirea numărului la 2 este diferit de 0 (folosind operatorul %).
4. Dacă restul nu este egal cu 0, numărul este impar și este afișat.

Observație (programare corectă): se verifică dacă lower ≤ upper. Dacă nu, intervalul este invalid și trebuie corectat sau semnalat.

C++ Începători

Variantă C++ echivalentă: citește limitele și afișează numerele impare.

Cod complet iostream + for + if 
#include <iostream>
using namespace std;

int main() {
    int lower, upper;

    cout << "Enter the lower limit for the range:";
    cin >> lower;
    cout << "Enter the upper limit for the range:";
    cin >> upper;

    for (int i = lower; i <= upper; i++) {
        if (i % 2 != 0) {
            cout << i << "\n";
        }
    }

    return 0;
}

Explicația programului

1) Utilizatorul introduce limitele intervalului.
2) Bucla parcurge intervalul și verifică imparitatea cu %.
3) Numerele impare sunt afișate.

Observație (programare corectă): se verifică dacă lower ≤ upper. Dacă utilizatorul le introduce invers, se poate corecta sau se poate afișa eroare.
