Program pentru afișarea tuturor numerelor impare dintr-un interval

3 variante: Python (începători), Python (funcții), C++ (începători).

Python Începători

Cod comentat (pentru claritate didactică).

Cod complet Python (începători) 
# Citim limita inferioară a intervalului (lower)
lower = int(input("Enter the lower limit for the range:"))

# Citim limita superioară a intervalului (upper)
upper = int(input("Enter the upper limit for the range:"))

# Parcurgem toate numerele din intervalul [lower, upper]
for i in range(lower, upper + 1):
    # Dacă numărul nu se împarte exact la 2, atunci este impar
    if i % 2 != 0:
        # Afișăm numărul impar
        print(i)

Explicatia programului

1. Utilizatorul trebuie să introducă limita superioară și limita inferioară ale intervalului.
2. Bucla for parcurge valorile de la limita inferioară până la limita superioară.
3. Expresia din instrucțiunea if verifică imparitatea folosind operatorul % (rest la împărțirea la 2).
4. Dacă restul nu este 0, numărul este impar și este afișat.

Observație (programare corectă): în practică se verifică dacă lower ≤ upper. Dacă utilizatorul introduce limitele invers (lower > upper), se poate interschimba valorile sau se poate afișa un mesaj de eroare.

Python Funcții

Variantă comentată: funcție + apel.

Cod complet Funcție + apel 
# Funcție care afișează toate numerele impare din intervalul [lower, upper]
def print_odd_numbers_in_range(lower, upper):
    # Parcurgem toate numerele din interval
    for i in range(lower, upper + 1):
        # Verificăm dacă i este impar (rest diferit de 0 la împărțirea la 2)
        if i % 2 != 0:
            # Afișăm numărul impar
            print(i)

# Citim limita inferioară și limita superioară
lower = int(input("Enter the lower limit for the range:"))
upper = int(input("Enter the upper limit for the range:"))

# Apelăm funcția pentru a afișa numerele impare din interval
print_odd_numbers_in_range(lower, upper)

Explicația programului

1. Se citește intervalul [lower, upper].
2. Funcția parcurge intervalul și verifică fiecare număr cu %.
3. Numerele impare sunt afișate.

Observație (programare corectă): se verifică dacă lower ≤ upper. Dacă nu, intervalul este invalid și trebuie corectat sau semnalat.

C++ Începători

Variantă comentată: citește limitele și afișează numerele impare.

Cod complet iostream + for + if 
#include <iostream>
using namespace std;

int main() {
    // Declarăm limitele intervalului
    int lower, upper;

    // Citim limita inferioară
    cout << "Enter the lower limit for the range:";
    cin >> lower;

    // Citim limita superioară
    cout << "Enter the upper limit for the range:";
    cin >> upper;

    // Parcurgem intervalul [lower, upper]
    for (int i = lower; i <= upper; i++) {
        // Dacă i este impar, îl afișăm
        if (i % 2 != 0) {
            cout << i << "\n";
        }
    }

    return 0;
}

Explicația programului

1) Se citesc limitele intervalului.
2) Bucla parcurge intervalul și verifică imparitatea cu %.
3) Numerele impare sunt afișate.

Observație (programare corectă): se verifică dacă lower ≤ upper. Dacă utilizatorul le introduce invers, se poate corecta sau se poate afișa eroare.
